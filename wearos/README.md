## Requirements
- Pixel Watch 3 (or compatible Wear OS 4 device)
- LTE connectivity (T-Mobile prepaid)
- Clawdbot Gateway accessible (local network or Tailscale)

## Features (Planned)
- Real-time notifications from Clawdbot
- Voice command input via Google Assistant
- Quick action tiles
- Dashboard view
- Standalone operation (no phone required)

## Development
Project structure will follow standard Android/Wear OS conventions:
- Kotlin + Jetpack Compose
- WebSocket client for gateway connection
- Foreground service for background connectivity
